# AIchat
 A TOOL to chat with AI
## 前言
本项目目的是容纳市面上的所有的LLM（大语言模型）或者本地大模型（ollama），并且提供大量的可自定义参数用于微调大模型，以用统一的架构来便利用户的使用。
