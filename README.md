# AIchat
 A TOOL to chat with AI
## 前言
这是一个练手的小项目，希望能通过捣鼓其中的细节，使我更加深入学习计算机
项目基础是一个可以调用api的ai小工具，可以自定义各种参数，以更加个性化的使用ai。
## 目前功能如下
1. 拥有基本的对话框架，对话框，人机对话显示，发送按钮
2. 拥有查看历史，新建历史功能
3. 更改api（目前只拥有种api，未来会有自建api的规划）
