FLYINGPIG-Chatbox 用户隐私政策
====================

最后更新日期：[2025.5.3]

感谢您使用 FLYINGPIG-Chatbox！我们重视您的隐私，并致力于保护您的个人信息。本隐私政策旨在说明我们如何收集、使用、存储和保护您的信息。请仔细阅读。

---
一、 我们收集的信息
---

FLYINGPIG-Chatbox 在您的本地计算机上运行，并收集以下类型的信息，这些信息也主要存储在您的本地计算机上：

1.  账户信息:
    *   用户名: 用于识别您的账户，关联您的聊天记录和设置。
    *   密码 (哈希后): 我们存储的是您密码经过安全哈希处理后的版本，用于验证您的登录，我们不存储您的原始密码。

2.  聊天内容:
    *   用户消息: 您在聊天中输入的内容。
    *   AI 回复: AI 模型返回的响应内容。
    *   会话信息: 包括会话 ID 和时间戳，用于组织和检索聊天记录。

3.  API 密钥
    *   如果您选择为特定的 AI 模型提供 API 密钥，这些密钥将与您的用户名关联并存储。

4.  配置信息:
    *   登录状态: 用于在您下次打开应用时保持登录。

---
二、 我们如何使用您的信息
---

我们使用收集到的信息来：

1.  提供和维护服务: 用于用户登录、身份验证、保存和加载聊天记录。
2.  处理聊天请求: 将您的消息发送给您选择的 AI 模型（通过本地运行的 `simple-one-api` 代理服务）以获取回复。
3.  管理 API 密钥: 将您提供的 API 密钥用于配置本地的 `simple-one-api` 服务，以便它能够代表您向相应的第三方 AI 服务进行身份验证。
4.  个性化体验: 加载您保存的 API 密钥和用户设置。

---
三、 数据存储和安全
---

1.  本地存储: 这是 FLYINGPIG-Chatbox 最核心的隐私特性之一。您的所有个人数据，包括账户信息 (`users.csv`)、完整的聊天记录 (`chat_history.csv`)、您保存的 API 密钥 (`user_api_keys.json`) 以及部分配置 (`Uconfig.json`)，均存储在您运行 FLYINGPIG-Chatbox 应用程序的本地计算机上。数据不会主动上传到我们的服务器。
2.  密码安全: 我们采用哈希算法处理您的密码，增强安全性。
3.  API 密钥安全: 您提供的 API 密钥存储在本地，并仅用于通过本地 `simple-one-api` 服务与相应的第三方 AI 模型进行交互。请您妥善保管您的 API 密钥，并了解您所使用的第三方 AI 服务提供商的隐私政策。您有责任确保运行 FLYINGPIG-Chatbox 的计算机环境的安全。

---
四、 数据共享
---

FLYINGPIG-Chatbox 本身不会将您的个人数据分享给除以下情况之外的任何第三方：

1.  与第三方 AI 服务提供商的交互:
    *   当您发送消息时，该消息内容以及（如果您为所选模型提供了）相应的 API 密钥，会通过本地运行的 `simple-one-api` 代理服务发送给您选择的第三方 AI 模型提供商（例如，智谱 AI、阿里云、Deepseek 等）以处理您的请求并生成回复。
    *   重要提示: 您与这些第三方 AI 服务提供商的交互受其各自隐私政策的约束。我们强烈建议您在使用需要 API 密钥的模型之前，查阅并理解对应服务提供商的隐私政策。FLYINGPIG-Chatbox 不对这些第三方服务的数据处理行为负责。

2.  无其他共享: 除上述情况外，FLYINGPIG-Chatbox 不会出售、交易或以其他方式转让您的个人身份信息给外部方。

---
五、 用户控制与选择
---

由于数据主要存储在本地，您拥有较高的控制权：

1.  管理 API 密钥: 您可以通过应用程序界面添加、更新或清除为您账户保存的 API 密钥。
2.  管理聊天记录和账户数据: 您可以直接在本地计算机上管理（例如，查看、备份或删除）存储用户数据的文件（`users.csv`, `chat_history.csv`, `user_api_keys.json`, `Uconfig.json`）。请注意，直接修改或删除这些文件可能会导致数据丢失或应用功能异常。

---
六、 儿童隐私
---

FLYINGPIG-Chatbox 不适用于 13 岁以下的儿童。我们不会有意收集 13 岁以下儿童的个人身份信息。

---
七、 本隐私政策的变更
---

我们可能会更新本隐私政策。如果我们做出更改，我们将在应用程序内或通过其他适当方式通知您，并更新政策顶部的"最后更新日期"。我们建议您定期查看本政策以了解最新信息。

---
八、 联系我们
---

如果您对本隐私政策有任何疑问，请通过 [https://github.com/Heyflyingpig/AIchat] 与我们联系。
