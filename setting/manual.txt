FLYINGPIG-Chatbox 用户操作手册
====================

欢迎使用 FLYINGPIG-Chatbox！本手册将引导您了解并使用本应用程序的各项功能。

---
一、 开始使用：注册与登录
---

1.  注册账户:
    *   首次使用时，您需要注册一个账户。
    *   在登录界面找到"注册"选项。
    *   输入您想要使用的用户名和密码。密码在发送前会在您的电脑上进行安全处理（哈希），服务器不会存储您的原始密码。
    *   点击"注册"按钮。如果用户名未被占用且信息有效，您将看到注册成功的提示。

2.  登录账户:
    *   在登录界面输入您已注册的用户名和密码。
    *   点击"登录"按钮。
    *   登录成功后，系统会自动加载您的个人设置（如已保存的 API 密钥）并进入主聊天界面。
    *   您的登录状态会被记录，下次打开应用时可能会自动登录。

---
二、 主聊天界面
---

1.  聊天窗口:
    *   界面中央是主要的聊天区域。
    *   您输入的消息会显示在一侧，AI 的回复会显示在另一侧。
    *   使用底部的输入框键入您想对 AI 说的话。
    *   按下回车键或点击"发送"按钮即可将消息发送给 AI。

2.  新建聊天:
    *   如果您想开始一个全新的对话主题，可以点击界面上方的"新建聊天"或类似按钮。
    *   这将生成一个新的会话 ID，当前聊天窗口内容会被清空，您可以开始新的对话，旧的对话会被保存在历史记录中。

---
三、 管理聊天历史
---

1.  查看历史记录:
    *   通常在界面的侧边栏（或通过特定按钮访问）会显示您的聊天历史列表。
    *   每个历史条目会显示该会话的部分起始内容和最后交互的时间。
    *   历史记录是按用户隔离的，您只会看到自己账户下的聊天记录。

---
四、 模型与 API 密钥管理
---

FLYINGPIG-Chatbox 支持连接多种不同公司或来源的大语言模型。部分模型可能需要您提供自己的 API 密钥才能使用。

1.  查看可用模型:
    *   在应用的设置区域（或专门的模型管理界面），您可以查看到当前支持的所有 AI 模型列表，通常按提供商分类（如智谱 AI、阿里云等）。
    *   每个模型旁边可能会有一个状态指示，告知您当前是否需要为该模型提供 API 密钥 (通常显示为 "需要 Key" 或类似提示)。
    *   "需要 Key" 的状态取决于：您是否已经为该模型保存了有效的密钥，以及该模型是否有可用的默认/共享密钥。如果两者都没有，则需要您提供。

2.  保存/更新 API 密钥:
    *   找到您想为其设置密钥的模型。
    *   点击该模型旁边的"设置 Key"、"添加 Key"或类似按钮。
    *   在弹出的输入框中，粘贴您的 API 密钥。
    *   点击"保存"或确认按钮。
        *   如果保存的是有效的密钥，系统会存储它，并可能自动将该模型设为当前选用模型。
        *   如果您保存一个空的密钥，系统会清除之前为该模型保存的密钥。
    *   保存密钥后，底层的 API 服务会自动重启以应用新的配置。

3.  选择当前使用的模型:
    *   在模型列表区域，您可以选择想要进行聊天的模型。
    *   点击您想使用的模型名称旁边的"选择"或类似按钮。
    *   注意: 如果您选择了一个标记为"需要 Key"但您尚未提供有效 Key的模型，那么后续的聊天请求可能会失败。请确保您已为需要 Key 的模型正确保存了 Key。
    *   选择模型后，后续您发送的消息将由该选定模型处理。

---
五、 其他设置
---

1.  查看用户协议/手册:
    *   设置菜单中通常包含查看"用户协议"和"用户操作手册"（即本文档）的选项。

---
六、 登出账户
---

*   如果您想退出当前账户，可以在设置或用户菜单中找到"登出"按钮。
*   点击登出后，系统会清除您的登录状态，并恢复默认的 API 配置。您将返回到登录界面。

---
